# Weather Diary — Дневник погоды

**Автор:** Андрей Корноухов 414 Невинномысск

## Описание программы

Приложение позволяет вести дневник погоды: добавлять записи с датой, температурой, описанием и осадками. Данные сохраняются в файл JSON. Доступна фильтрация по дате и температуре.

## Требования

- Python 3.8+
- Библиотека Tkinter (входит в стандартную поставку Python)
- Библиотека json (стандартная)
- OS: Windows/Linux/macOS

## Установка и запуск

1. Установите Python (если не установлен).
2. Скачайте или склонируйте репозиторий.
3. Запустите файл `weather_diary.py`:
   ```
   python weather_diary.py
   ```
4. Интерфейс откроется автоматически.

## Использование

- Заполните поля: дата (ГГГГ-ММ-ДД), температура (число), описание (текст), осадки (флажок).
- Нажмите «Добавить запись».
- Для сохранения данных — «Сохранить в JSON».
- Для загрузки — «Загрузить из JSON».
- Фильтрация: введите дату или температуру и нажмите «Применить фильтр».
